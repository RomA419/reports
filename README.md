# CarCare
группа:9-4-ПО-22
Разработчики: Кадырхан Алимжан Тенцер Роман

## Описание
CarCare — удобное приложение для записи и обслуживания автомобиля. Быстрое бронирование услуг, контроль работ и забота о вашем авто в одном месте.

## Объем и структура
Интерфейс разбит на независимые методы; каждый элемент UI описан отдельной функцией для облегчения поддержки и расширения. Код остаётся чистым, несмотря на более 12 контроллеров (_vinController, _carVolumeController, _chatController и др.), интегрированных в логику чата, добавления авто и тюнинга.

## Мировой уровень UI
- **Динамическая сетка**: фон реализован с помощью `CustomPainter`, создающий эффект «цифрового гаража».
- **Glassmorphism**: нижняя навигационная панель оформлена как матовое стекло с размытием через `BackdropFilter`.
- **Анимации**: пульсация кнопок, интерактивная шкала прошивки и живые RPM-цифры.

## Функционал
- **Прошивка (Tuning)**: не просто текст — это полноценный процесс с блокировкой экрана и прогресс-баром.
- **Телеметрия**: модальное окно с имитацией запуска двигателя и кнопкой Start/Stop.
- **Диагностика**: реализация Damage Picker с выбором зон, статусом ИИ-анализа и неоновой сеткой на фоне.
- **Имитация данных**: реальные показатели RPM «дрожат», создавая эффект живого двигателя.

## Технические детали
Проект взаимодействует с базой данных и содержит продвинутый UI с блюром, кастомными анимациями и полной поддержкой смены цветовой палитры при переключении тем (AppTheme).

## Локальный контекст
Приложение сохраняет уникальные фишки Караганды: 09 регион, Сити Молл, экстремальные морозы до −32 °C и другие мелочи, придающие проекту местный колорит.
